=========================================
SAMPLE ALGORITHM: MORNING GREETING
=========================================

Algorithm Name: Morning Greeting Algorithm

STEPS:
1. Check the time
2. IF time is before 12:00
   THEN say "Sawubona! Good morning!"
3. ELSE IF time is before 18:00
   THEN say "Good afternoon!"
4. ELSE
   THEN say "Good evening!"
5. Smile and be friendly
6. END


=========================================
SAMPLE ALGORITHM: HELPING A FRIEND
=========================================

Algorithm Name: Ubuntu Helper Algorithm

STEPS:
1. Observe your friend
2. IF friend looks sad or needs help
   THEN ask "Are you okay? Can I help?"
3. IF friend says yes
   THEN listen carefully
   THEN offer help
4. ELSE
   THEN say "I'm here if you need me"
5. Stay kind and supportive
6. END

REMEMBER: This is Ubuntu in action!


=========================================
SAMPLE ALGORITHM: LED PATTERN
=========================================

Algorithm Name: Blinking LED Pattern

STEPS:
1. Turn LED ON
2. Wait 1 second
3. Turn LED OFF
4. Wait 1 second
5. REPEAT steps 1-4 five times
6. END
